广告位置说明：

1、图片广告位的宽和高单位为像素（px）

2、图片广告位的宽和高应填写大于0的整数，否则可能影响图片的正常显示

3、文字广告位对宽和高没有限制

4、广告位名称非技术人员请谨慎修改，可能会影响前台图片的调用
